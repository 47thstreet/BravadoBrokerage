Introduction
Embed iFrame script code
To implement the listing widget on your site, we recommend creating a container with 100% width (of the entire viewport). Some elements of the widget such as the split view, map view & listings details pages work & look best on full screens.

The following code snippets provide you with various examples for integrating the RealtyMX IDX IFrame into your website.

Please review the available parameters & option values provided and adjust the widget parameters for your requirements.

You may add any of the supported parameters below on the script tag using the following format:

data-parametername="value"

A parameter should be defined no more then once per tag.

Languages
We've included native multi language support for the major elements (search, search results, listings details). The translated dictionary is still being updated. If you find any mistakes or have suggestions for specific words or phrases, then please let us know.

We currently support Japanese (JP), Chinese (CH) & Korean (KR).

Set data-lang to a language code such as data-lang="jp".

URL Access Guide
Accessing Listings Directly via your URL
The following instructions allow you to access listing or create direct links to pages within the iframe using the page on your domain name that contains the IFRAME widget.

Display your Search Landing Page
yourdomain.com/your-listings-page/#listings - Directs the IFRAME to your Default search landing page.

Display A Company Listing
yourdomain.com/your-listings-page/#listing-LISTINGID - Directs the IFRAME to a Company listing details page.

Invoke The Login Pop-Up
yourdomain.com/your-listings-page/#login - open the pop-up modal window

Invoke The Sign-Up Pop-Up
yourdomain.com/your-listings-page/#signup - open the pop-up modal window

Quicksearch Widget
Add a stand-alone IDX "quicksearch" widget to your home page or landing pages that can search and load the IDX listing results on your site.

Requirements:
Source Element Container ID
sourceID Required - valid DOM element ID selector to put the search box in.

Target URL OR Container ID Require One
targetID - Enter a element ID selector on the current page to load the IFRAME within.
targetURL - Enter a valid URL for your IFrame's search page (e.g. https://yourdomain.com/your-listings-page).

 Live Preview Example
Target Page Example:

This example will allow you to add this widget to one page, and allow you to target another URL on your domain to render the results.


    <script data-client_id="1462" 
        data-page="quicksearch" 
        data-sourceID='realtymx_idx1' 
        data-targetURL="https://idx.realtymx.com/widget/target.cfm"
        src="https://idx.realtymx.com/widget/idx_qsearch.js">
    </script>
    <div id="realtymx_idx1"></div>
    
Element ID Usage Example:
This example will allow you to add both this quicksearch widget and a destination container ID to render the full IDX IFrame on the same page.


    <script data-client_id="1462" 
        data-page="quicksearch" 
        data-sourceID='realtymx_idx1' 
        data-targetID="realtymx_idx2"
        src="https://idx.realtymx.com/widget/idx_qsearch.js">
    </script>
    <div id="realtymx_idx1"></div>
    <div id="realtymx_idx2"></div>
